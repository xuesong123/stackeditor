## 概述

StackEditor 是一个基于 ProseMirror 的纯原生JS的富文本编辑器。

### 特点和功能

1. 基于 ProseMirror 实现。
2. 纯原生 JS 实现，不依赖其他前端框架。
3. 内置 Placeholder 支持
4. 内置标题框支持
5. 支持类似 WORD 文档的模式
